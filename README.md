# Компонент Bitrix: Форма отзывов (custom:review.form)

Компонент предназначен для сбора отзывов пользователей, сохранения их в инфоблок и отображения после модерации.

## Особенности

- Добавление отзывов с полями: **Имя, Плюсы, Минусы, Отзыв**.
- Все поля настраиваются через свойства инфоблока.
- Валидация формы на стороне клиента (JavaScript) и на сервере (PHP).
- Отзывы отображаются **только после модерации** (`ACTIVE = N` по умолчанию).
- Используется **D7 ORM** для работы с элементами инфоблока.

## Установка

Создать инфоблок  Отзывы (символьный код:reviews, API код:Reviews)  
Добавить свойства:
Плюсы - строка - PLUS
Минусы - строка - MINUS

Скопировать компонент в папку:
/local/components/custom/review.form/

Подключение компонента:
<?php
$APPLICATION->IncludeComponent(
    "custom:review.form",
    "",
    [
        "IBLOCK_ID" => 5,
        "REVIEWS_COUNT" => 10,
        "SUCCESS_MESSAGE" => "Спасибо! Ваш отзыв отправлен на модерацию.",
        "CACHE_TIME" => 3600,
    ]
);
?>
